# Probatio diabolica

A testing framework with large language models (LLMs) in Ruby.
Not ready for production use.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'probatio_diabolica'
```

And then execute:

```bash
bundle install
```

## Usage

```ruby
require 'probatio_diabolica'

RSpec.describe 'Probatio Diabolica' do
  it 'does something' do
    expect(true).to eq(true)
  end
end
```
